Lisez-moi

Présentation

ContestOrg est un logiciel d'organisation de tournois. Pour plus d'informations, rendez-vous :

  • dans l'aide en cliquant ici
  • sur le site web en cliquant ici

Vous pouvez également vous rendre sur l'espace de développement Google Code en cliquant ici.

Lancer ContestOrg

Sous Windows : double-cliquer sur ContestOrg.exe

Sous Linux et Mac-OS :

  • Vérifier si Java est installé : exécuter java -version ce qui doit afficher la version de Java s'il est installé
  • Installer Java (s'il n'est pas déjà installé) : suivre la procédure d'installation en fonction de votre plateforme
  • Exécuter java -jar ContestOrg.jar (depuis le répertoire de ContestOrg)
